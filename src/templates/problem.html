Problem: Palindrome

A palindrome is a word that is read the same way from left to right and vice versa.

Given a word S, return a palindrome formed from all the letters of S. If more than one palindrome can be formed from the letters of S, find the lexicographically smallest. If no palindrome can be formed from the letters of S, output "IMPOSSIBLE" (without quotes, of course).

Input

On the first and only line stands the word S, consisting of lowercase Latin letters.

Output

On the first and only line output the found palindrome or "IMPOSSIBLE" if such does not exist.

Limitations

TL 0.5S
ML 32MB
1 ≤ |S| ≤ 106

Examples

Example Input Example Output
abaaba aabbaa
ab IMPOSSIBLE
